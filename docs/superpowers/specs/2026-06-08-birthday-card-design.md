# 生日电子贺卡 - 设计说明

## 概述

单页面生日电子贺卡，移动端优先，二维码扫码进入。活泼欢快风格 + 圆润卡通艺术字。

## 页面结构（三段式，纵轴）

```
┌─────────────────────┐
│  生日快乐标语区      │ ← 蛋糕阶段显示
├─────────────────────┤
│                     │
│  祝福文字 / 蛋糕舞台 │ ← 核心动画区域
│                     │
├─────────────────────┤
│  滑动提示区          │ ← 固定底部
└─────────────────────┘
```

## 动画序列

### 阶段一：祝福交替
- 6 条浪漫生日祝福，每次展示一条
- 用户上滑（或鼠标滚轮向下）推进，下滑（或滚轮向上）可回退
- 切换动画：弹性淡入 + 位移（`spring-like ease-out`）
- 底部显示上滑箭头提示
- 最后一条祝福滑过后进入阶段二

### 阶段二：蛋糕降临
- **禁止滑动**，全自动播放
- 物品掉落顺序（弹性缓出动画）：
  1. 盘子从上方掉落，中间位置停住
  2. 蛋糕体掉落
  3. 奶油层掉落
  4. 蜡烛掉落
  5. 火焰点燃（`opacity` + 缩放 + 闪烁动画）
- 蜡烛点燃后：
  - 顶部 "生·日·快·乐" 四个字逐个弹入
  - 背景彩带/气球飘落
  - 生日快乐歌响起

## 技术方案

| 项 | 选择 |
|----|------|
| 结构 | 单文件 HTML（内嵌 CSS + JS） |
| 动画 | CSS `@keyframes` + `transform`，GPU 加速 |
| 音频 | Web Audio API 合成生日歌（避免移动端自动播放限制） |
| 滑动 | JS touch 事件 + wheel 事件，全屏滚动锁定 |

## 视觉规格

### 配色
- 主背景：深紫 `#1a0033` → 粉色 `#ff6b9d` 渐变
- 祝福文字：白色填充 `#fff`，粉色描边 `#ff3388`，多层立体阴影
- 蛋糕：暖黄 `#f5d47a` 蛋糕体 / 白 `#fffef5` 奶油 / 彩色条纹蜡烛
- 标语：金色渐变 `#ffd700` → `#ff8c00`，每个字独立动画
- 彩带：粉 `#ff6b9d`、黄 `#ffd700`、蓝 `#66ccff`、绿 `#66ff99`

### 艺术字（圆润卡通风格）
- Web 字体：Google Fonts - "ZCOOL KuaiLe"（中文自带圆润卡通感）
- 祝福字：白色 + 多层 `text-shadow`（描边 + 立体凸起）+ `letter-spacing: 0.08em`
- 标语字：金色渐变填充 + 4px 描边 + 多层阴影叠加，每个字 `font-size: 3rem`
- 备选 fallback：`"Microsoft YaHei", sans-serif`

### 蛋糕（CSS 绘制 + 动画）
- 盘子：椭圆 `border-radius: 50%`，灰白色，带阴影
- 蛋糕体：矩形 `border-radius` 圆角
- 奶油：波浪形边缘（`border-radius` + `::before`/`::after` 装饰）
- 蜡烛：细长矩形，彩色条纹，顶部火焰 `box-shadow` 模拟
- 火焰：CSS 圆角三角形 + `@keyframes` 闪烁

### 生日快乐歌（Web Audio API）
- 简谱：`C D C F E ...`（Happy Birthday To You）
- 使用 `OscillatorNode` 正弦波 + 包络控制
- BPM 120，每个音符约 250ms
- 歌曲总时长约 10 秒，不循环

## 交互流程

```
用户扫码 → 页面加载 → 第1条祝福弹出
    ↓
用户滑动 → 第N条祝福（淡入淡出）
    ↓
用户滑动 → 最后一条 → 自动进入蛋糕阶段
    ↓
蛋糕动画播放 → 蜡烛点燃 → 歌曲响起
    ↓
标语弹出 → 彩带飘落 → 停留在完成画面
```

## 非功能性需求
- 首屏加载 < 2s（所有资源内嵌，无外部网络依赖）
- 60fps 动画
- 适配 iPhone SE ~ iPhone 15 Pro Max（320px ~ 430px 宽度）
- 同时适配桌面端（居中展示，限制最大宽度 480px）
